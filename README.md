# VueJs Boilerplate - Babel / Webpack / Sass

![Banner](http://i.imgur.com/G2Uqw43.png)

#### TECHNOLOGIES

* VueJs
* GSAP - TweenMax
* Babel | ES2015
* Webpack
* ESLint
* ESDoc
* SASS

##### Install dependancies :
```shell
  npm i
```

##### Launch the project :
```shell
  npm start
```

The project will be launched at http://localhost:3000/

#### Build for production :
```shell
  npm run build
```
